Framework Features

What makes Griddock special

🎯 CSS Grid Based

Built entirely with CSS Grid instead of Flexbox, providing more powerful and flexible layouts for modern web development.

📱 Fully Responsive

Mobile-first responsive design with breakpoint utilities and responsive grid classes for all screen sizes.

🎨 Customizable

CSS custom properties for easy theming and customization. Change colors, spacing, and more with simple variable updates.

🚀 No JavaScript

Pure CSS framework that works without any JavaScript dependencies. Just include the CSS and start building.

🔧 Developer Friendly

Intuitive class naming inspired by Bulma, with comprehensive utility classes for rapid development.

⚡ Lightweight

Optimized CSS with minimal overhead. Clean, efficient code that loads fast and performs well.
